# 📊 Power BI Sales Dashboard

## 📌 Project Overview
This project presents an interactive Sales Dashboard built using Microsoft Power BI. The dashboard provides meaningful insights into sales performance, customer behavior, and product trends through dynamic visualizations and KPIs.

## 🎯 Objectives
- Analyze overall sales performance.
- Track revenue, profit, and orders.
- Identify top-performing products and categories.
- Understand customer purchasing patterns.
- Enable data-driven business decisions.

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel (Dataset)

## 📈 Dashboard Features
- KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders
- Sales Trend Analysis
- Category-wise Sales
- Region-wise Performance
- Customer Analysis
- Interactive Filters (Slicers)
- Dynamic Charts and Visualizations
